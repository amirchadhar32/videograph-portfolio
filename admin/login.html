First-time Firebase setup (fixes auth/configuration-not-found):
  1. Open Firebase → Authentication → Sign-in method
  2. Click Email/Password → turn Enable ON → Save
  3. Go to Users tab → Add user (your admin email + password)
  4. Authentication → Settings → Authorized domains → add your site (e.g. localhost, braincoredev.netlify.app)
